# FinTrack
A finance tracker is a tool that monitors income, expenses, and budgets to provide a clear picture of your financial health.

Finance Tracker

A modern, lightweight, and user-friendly Expense Tracker Web Application designed to help users efficiently manage their daily finances. Track income, monitor expenses, and maintain a clear overview of your financial health — all in real time.

🚀 Features

✨ Core Functionalities

➕ Add income and expense transactions seamlessly
❌ Delete transactions instantly
🔄 Real-time UI updates after every action
📅 Automatic date generation for each transaction
⌨️ Enter key support for faster input

📊 Financial Insights

💵 Total Income calculation
💸 Total Expense tracking
🧮 Net Balance overview

👥 Smart Utilities

🤝 Split expenses with friends
📌 Latest transactions displayed first

💾 Storage

Local data persistence using LocalStorage (No backend required)
🛠️ Tech Stack
Technology	Purpose
HTML	Structure
CSS	Styling & Layout
JavaScript	Logic & Interactivity
LocalStorage	Data Persistence (Browser)

💡 How It Works
All transactions are stored in the browser using LocalStorage
Data is converted into string format using JSON.stringify() before saving
Data is retrieved and converted back using JSON.parse()
The UI dynamically updates after every transaction (add/delete)
